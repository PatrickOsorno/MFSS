Ayuda




En esta sección el cliente registra sus datos en el sistema, primero, se solicitan los datos personales (nombre, correo, etc.), posteriotmente sus direcciones (Dirección principal y dirección de entrega), por último se solicita el horario de entrega, y toda acción se completa con el boton guardar de cada sección.

Muy importante, se debe respetar el flujo de registro, primero los datos del cliente, luego las direcciones, y por último el horario, en caso contrario no se permitira el registro y el sistema mostrara mensajes de error.
